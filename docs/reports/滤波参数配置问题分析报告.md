# 📋 滤波参数配置问题分析报告

**文档编号**: TECH-REPORT-2026-04-13
**项目**: 创源启明蝴蝶四翼开源代码
**版本**: V6.0 仿生优化版
**作者**: hanchan
**发布日期**: 2026年04月13日

## 🎯 问题概述

**现象**: 系统在运行过程中出现角度数据异常、电机抖动、控制精度下降等问题。

**根本原因**: AS5600磁编码器滤波参数配置不当，限幅滤波阈值与低通滤波系数不匹配，导致滤波效果不佳。

## 🔍 问题定位

### 1. 问题代码位置

**文件**: `user/module/AS5600/AS5600_PWM.c`
**行号**: 84-91

```c
/**
 * @brief 限幅滤波阈值（编码器单位）
 * 
 * 作用：防止ADC读数突变（可能是干扰或接触不良）
 * 如果两次采样的差值超过此阈值，则丢弃新值，使用旧值
 * 
 * ⭐【V6.3针对>100噪声优化】
 * 原值150对于>100的噪声太宽松，已收紧到80
 * 
 * ⚠️ 权衡说明：
 *   - 电机正常运动时每2ms的变化量：
 *     - 低速（<3Hz）：约20-50编码器值/2ms ✅ 安全
 *     - 中速（3-5Hz）：约50-80编码器值/2ms ✅ 临界
 *     - 高速（>5Hz）：约80-120编码器值/2ms ⚠️ 可能偶尔丢弃
 *   - 但是！被丢弃的采样会被低通滤波平滑，影响很小
 *   - 相比之下，>100的噪声导致的抖动问题更严重
 * 
 * 可调范围：50 ~ 200
 * 当前值：80（严格版，强噪声抑制）
 * 建议：
 *   - 高速扑动(>8Hz)数据断续 → 增大到100-120
 *   - 仍有明显尖峰噪声 → 减小到60
 *   - 一般保持80即可（针对当前AS5600编码器）
 */
#define LIMIT_THRESHOLD 200
```

### 2. 问题分析

**核心矛盾**：
- **注释建议值**: 80（严格版，强噪声抑制）
- **实际定义值**: 200（过于宽松）
- **采样频率**: 500Hz（每2ms一次）
- **低通滤波系数**: 0.2f

**参数关联性分析**：

| 参数 | 当前值 | 影响 | 建议值 |
|------|--------|------|--------|
| 采样间隔 | 2ms (500Hz) | 控制数据更新频率 | 2ms (保持) |
| 限幅滤波阈值 | 200 | 过于宽松，无法有效滤除噪声 | 80-100 |
| 低通滤波系数 | 0.2f | 中等平滑效果 | 0.15-0.2 |

**频率与滤波的关系**：
- **低速扑动** (<3Hz): 每2ms变化20-50编码器值 → 限幅阈值80足够
- **中速扑动** (3-5Hz): 每2ms变化50-80编码器值 → 限幅阈值80临界
- **高速扑动** (>5Hz): 每2ms变化80-120编码器值 → 限幅阈值需100-120

## 📊 影响范围

### 1. 功能影响
- ⚠️ **角度数据异常**: 噪声未被有效过滤，导致角度数据抖动
- ⚠️ **电机控制不稳定**: PID控制器接收到异常数据，导致电机抖动
- ⚠️ **高速性能下降**: 高速扑动时可能出现数据断续
- ⚠️ **系统响应滞后**: 滤波参数不匹配导致数据处理延迟

### 2. 性能影响
- **CPU负载**: 正常（500Hz采样率合理）
- **内存使用**: 正常（滤波算法内存占用低）
- **实时性**: 受影响（滤波延迟增加）

## 🛠️ 解决方案

### 1. 参数优化建议

**方案A - 平衡版**（推荐）：
- **限幅滤波阈值**: 100
- **低通滤波系数**: 0.18f
- **适用场景**: 中高速扑动（3-8Hz）

**方案B - 噪声抑制版**：
- **限幅滤波阈值**: 80
- **低通滤波系数**: 0.15f
- **适用场景**: 噪声较大环境，低速扑动

**方案C - 高速响应版**：
- **限幅滤波阈值**: 120
- **低通滤波系数**: 0.22f
- **适用场景**: 高速扑动（>8Hz）

### 2. 代码修改建议

```c
// 修改前
#define LIMIT_THRESHOLD 200
#define ALPHA_FLOAT 0.2f

// 修改后（方案A - 平衡版）
#define LIMIT_THRESHOLD 100
#define ALPHA_FLOAT 0.18f
```

### 3. 验证方法

1. **静态测试**:
   - 使用示波器观察ADC原始波形
   - 检查滤波前后数据对比

2. **动态测试**:
   - 低速扑动（2Hz）测试
   - 中速扑动（5Hz）测试
   - 高速扑动（10Hz）测试

3. **性能指标**:
   - 角度数据抖动范围 < ±10编码器值
   - 电机运行平稳无明显抖动
   - 高速扑动时数据连续无断续

## 📈 预期效果

| 测试项 | 优化前 | 优化后 |
|--------|--------|--------|
| 噪声过滤效果 | 差（噪声>100） | 好（噪声<30） |
| 高速响应 | 断续 | 连续 |
| 电机稳定性 | 抖动明显 | 运行平稳 |
| 控制精度 | 偏差大 | 偏差小 |

## 📌 注意事项

1. **参数联动性**: 限幅滤波阈值与低通滤波系数必须配合调整
2. **硬件差异**: 不同AS5600传感器的噪声特性可能不同，需要根据实际情况微调
3. **温度影响**: 温度变化可能影响传感器性能，建议在不同温度下测试
4. **机械振动**: 机械振动可能引入额外噪声，需要考虑机械结构优化

## 🎯 结论

滤波参数配置是影响系统性能的关键因素，需要根据实际运行条件进行精细调整。建议采用方案A（平衡版）作为默认配置，然后根据具体应用场景进行微调。

**最终建议**: 将限幅滤波阈值从200调整为100，低通滤波系数保持0.2f或调整为0.18f，以获得最佳的噪声抑制效果和高速响应性能。
