### 3️⃣ [双电机版本修改总结](./reports/双电机版本修改总结.md)

**分支名称**: `双电机版本`  
**修改日期**: 2026-04-10  
**修改目的**: 将4电机系统改为2电机系统，释放PA2/PA3引脚用于USART2接收机通讯

**核心变更**:
- ✅ 串口配置: USART1 (PB6/PB7) → USART2 (PA2/PA3)
- ✅ 电机数量: 4电机 → 2电机 (M1右前 + M3左前)
- ✅ 删除电机: M2(右后) + M4(左后)
- ✅ 编译状态: 通过 ✅

**详细清单**:
1. 串口配置修改（usart.c/h, stm32g0xx_it.c, main.c, elrs.c）
2. 电机控制修改（motor.c/h）
3. 飞行控制修改（flight_control.c）
4. 编码器配置修改（AS5600_PWM.c/h）
5. 定时器配置修改（tim.c, main.h）

**双电机引脚映射**:

| 功能 | STM32引脚 | 外设 |
|------|---------|------|
| USART2_TX | PA2 | 接收机TX |
| USART2_RX | PA3 | 接收机RX |
| M1(右前) PWM | PA15 + PB3 | TIM2 CH1/CH2 |
| M3(左前) PWM | PB4 + PB5 | TIM3 CH1/CH2 |
| M1(右前) ADC | PA1 | AD_CH1 |
| M3(左前) ADC | PA0 | AD_CH0 |
